Conversational RAG Chatbot with PDF Intelligence

An AI-powered conversational chatbot that allows users to upload PDF documents and ask context-aware questions using Retrieval-Augmented Generation (RAG).

* Features
1. Upload and chat with multiple PDF files
2. Conversational memory with chat history
3. Semantic document retrieval using embeddings
4. ChromaDB vector database integration
5. Groq LLM integration for fast response generation
6. Streamlit-based interactive UI
7. Context-aware question answering
8. Retrieval-Augmented Generation (RAG) pipeline

* Tech Stack
1. Python
2. LangChain
3. Streamlit
4. ChromaDB
5. Hugging Face Embeddings
6. Groq LLM
7. FAISS
8. PyPDFLoader

* Project Workflow
1. Upload PDF documents through the Streamlit interface.
2. Extract text from uploaded PDFs.
3. Split documents into smaller chunks.
4. Generate vector embeddings using Hugging Face models.
5. Store embeddings in ChromaDB.
6. Retrieve relevant document chunks based on user queries.
7. Generate accurate conversational responses using Groq LLM.
